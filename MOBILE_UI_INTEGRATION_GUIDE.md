# 手机端UI集成指南

## 📋 概述

本文档说明如何将新功能（环境管理、分类管理、人脸审核）集成到现有的 `mobile.html` 文件中。

## 🔧 集成步骤

### 方法1：自动集成（推荐）

运行自动集成脚本：
```bash
python integrate_mobile_ui.py
```

该脚本将自动完成所有集成步骤。

### 方法2：手动集成

#### 步骤1：添加新页面HTML

在 `templates/mobile.html` 文件中，找到 `<!-- ===== Settings Page ===== -->` 部分（约第609行），在该页面的 `</div>` 结束标签之前，添加以下内容：

```html
<!-- 在第695行之后（设置页面结束之前）添加 -->

<!-- ===== 环境管理页面 ===== -->
<div class="page" id="pageEnvironments">
    [复制 mobile_new_features.html 中的环境管理页面代码]
</div>

<!-- ===== 分类管理页面 ===== -->
<div class="page" id="pageCategories">
    [复制 mobile_new_features.html 中的分类管理页面代码]
</div>

<!-- ===== 人脸审核页面 ===== -->
<div class="page" id="pageFaceApproval">
    [复制 mobile_new_features.html 中的人脸审核页面代码]
</div>
```

#### 步骤2：添加弹窗HTML

在 `<!-- ===== Person Modal ===== -->` 部分（约第726行）之后，添加新的弹窗：

```html
<!-- 在人员编辑弹窗之后添加 -->

<!-- ===== 环境编辑弹窗 ===== -->
[复制 mobile_new_features.html 中的环境编辑弹窗代码]

<!-- ===== 分类编辑弹窗 ===== -->
[复制 mobile_new_features.html 中的分类编辑弹窗代码]

<!-- ===== 人脸审核弹窗 ===== -->
[复制 mobile_new_features.html 中的人脸审核弹窗代码]
```

#### 步骤3：添加CSS样式

在 `</style>` 结束标签之前（约第500行），添加新样式：

```css
/* 新增样式 */
.btn-small { ... }
.btn-icon { ... }
.list-item { ... }
/* ... 其他样式 */
```

#### 步骤4：添加JavaScript代码

在 `<script>` 标签内的末尾（约第1380行），添加新的JavaScript函数：

```javascript
// ========== 环境管理 ==========
function loadEnvironments() { ... }
function showAddEnvironmentModal() { ... }
// ... 其他函数

// ========== 分类管理 ==========
function loadCategories() { ... }
// ... 其他函数

// ========== 人脸审核 ==========
function loadPendingFaces() { ... }
// ... 其他函数

// ========== 页面初始化 ==========
document.addEventListener('DOMContentLoaded', function() {
    // ... 初始化代码
});
```

#### 步骤5：在设置页面添加入口

在设置页面的"系统"卡片之后（约第685行），添加新功能管理卡片：

```html
<div class="card">
    <div class="card-title">新功能管理</div>
    <div class="setting-item" onclick="switchPage('Environments')">
        <div>
            <div class="setting-label">环境管理</div>
            <div class="setting-desc">管理多个签到场景</div>
        </div>
        <button class="btn-icon">➔</button>
    </div>
    <!-- ... 其他菜单项 -->
</div>
```

## ✅ 验证集成

完成集成后，启动系统并验证：

1. 登录移动端管理界面
2. 进入"设置"页面
3. 应该能看到"新功能管理"卡片
4. 点击各个功能，验证页面可以正常打开和操作

## 🎯 功能说明

### 1. 环境管理
- **位置**：设置 → 环境管理
- **功能**：
  - 创建/编辑/删除环境
  - 设置环境的签到规则
  - 设置默认环境
  - 启用/禁用环境

### 2. 分类管理
- **位置**：设置 → 分类管理
- **功能**：
  - 创建/编辑/删除分类
  - 四级分类体系（集团→公司→部门→小组）
  - 树形展示
  - 拖拽排序

### 3. 人脸审核
- **位置**：设置 → 人脸审核
- **功能**：
  - 查看待审核人脸照片
  - 批准/拒绝人脸
  - 查看人脸详细信息
  - 添加拒绝原因

## 🐛 故障排除

### 问题1：新功能卡片不显示
- 检查是否正确添加了HTML代码
- 检查浏览器控制台是否有JavaScript错误
- 确认页面初始化代码已执行

### 问题2：点击菜单无响应
- 检查 `switchPage()` 函数是否正确扩展
- 确认新页面的ID格式正确（`pageEnvironments`）
- 检查JavaScript函数是否已定义

### 问题3：数据加载失败
- 检查API端点是否正常工作
- 查看浏览器网络请求是否成功
- 确认数据库升级已完成

## 📞 技术支持

如遇到问题，请检查：
1. 数据库是否已升级（运行 `database_migration_v2.py`）
2. API服务器是否正常运行
3. 浏览器控制台错误信息
4. 网络请求响应内容
