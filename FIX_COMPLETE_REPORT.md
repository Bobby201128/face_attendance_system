## 📱 系统使用流程

### 新的简化流程

**人员注册**:
1. 手机端 → 人员管理 → 添加人员
2. 手机端 → 拍照上传人脸
3. **自动激活** → 立即可用 ✅

**环境选择**:
1. PC端启动 → 显示环境选择对话框
2. 选择环境 → 应用该环境的签到规则

**签到识别**:
1. PC端启动摄像头 → 自动识别
2. 自动签到/签退 → 记录保存
